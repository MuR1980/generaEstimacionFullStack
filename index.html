Base de estimacion

Bajo
Cambio parametrico o ajuste pequeno, impacto localizado.
Medio
Modifica reglas, mapeos o UI con prueba funcional acotada.
Alto
Atraviesa integraciones, datos o procesos con dependencias.
Extra alto
Diseno nuevo, impacto transversal, migracion o rendimiento critico.

Los dias son rangos base usados como semilla de comunidad/practica de implementacion. La calibracion final debe usar historico interno, ambientes, aprobaciones y pruebas E2E.

1 dia = 8 horas. Distribucion: Diseno 35%, Documentacion 15%, Pruebas Unitarias 30% y Release 20%.
